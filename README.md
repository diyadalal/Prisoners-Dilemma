# Prisoner’s Dilemma Simulation

A Java-based Prisoner’s Dilemma simulation built using a reusable MVC and SimStation framework. The project models strategic interactions between agents using classic Prisoner’s Dilemma strategies and visualizes the simulation through a Swing-based GUI.

## Features
- Multiple strategies (Cooperate, Cheat, Tit-for-Tat, Random)
- Reusable MVC + SimStation architecture
- Swing-based graphical interface
- Start and statistics controls

## Technologies
Java, Swing, MVC, Object-Oriented Design

## How to Run
1. Open the project in an IDE 
2. Ensure the shared MVC / SimStation framework folder is included in the project
3. Navigate to  
   [`PrisonersDilemmaSimulation.java`](src/simulation/PrisonersDilemmaSimulation.java)
4. Run `PrisonersDilemmaSimulation.java`
5. Use the GUI menu to start the simulation and view statistics
